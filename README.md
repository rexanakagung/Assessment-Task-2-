# Assessment-Task-2-

### Description
This task requires me to plan, design and throughourly research a game called Top Trumps. Cars are used as Card Characters by using real life data from carsales.com. Top Trumps is a fast-paced card game where players use their deck of themed cards to out-score and "trump" their opponents. The goal is to collect every single card in the deck by winning rounds based on the numerical statistics (or attributes) printed on the cards.

### Features
Displays in game interface 

Displays Steps on how to play 

Displays Designs of cards 

Shows Diagrams of how the game works
